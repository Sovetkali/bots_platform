## [Unreleased]

### Добавлено
- Система управления пользователями с регистрацией и связью с ботами
- Модели базы данных: User, Bot, UserBot для связи многие-ко-многим
- Репозитории для работы с пользователями и ботами (UserRepository, BotRepository, UserBotRepository)
- UserService для бизнес-логики регистрации пользователей
- UserMiddleware для внедрения сервиса пользователей в обработчики
- Тесты для системы регистрации пользователей
- Утилита проверки состояния миграций базы данных
- Фильтры для обработки сообщений (IsPrivateChat)
- Middleware для исключения ботов из обработки (IsBotMiddleware)
- Утилитарная функция для извлечения пользователя из обновления Telegram
